Política de Privacidade — Fluxa

Última atualização: 29/01/2026

O Fluxa — seu dinheiro, no ritmo certo respeita a sua privacidade e está comprometido com a proteção dos dados pessoais, conforme a Lei Geral de Proteção de Dados (LGPD).

1. Dados coletados

  • E-mail informado pelo usuário
  • Dados financeiros inseridos manualmente
  • Dados técnicos básicos do aplicativo

2. Uso dos dados

Os dados são utilizados para funcionamento, identificação do usuário e melhoria da experiência.

3. Compartilhamento

O Fluxa não vende dados pessoais. Utiliza serviços de terceiros como Google AdMob para exibição de anúncios.

4. Anúncios

O aplicativo pode exibir anúncios personalizados ou não personalizados através do Google AdMob.

5. Direitos do usuário

O usuário pode solicitar acesso ou exclusão dos seus dados através do e-mail de suporte informado no aplicativo.
